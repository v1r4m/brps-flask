## quick start
### on Windows (CMD / PowerShell)
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```
### on Mac / Linux (bash/zsh)
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```
* It will be served on `localhost:5001`.
